# Melodex
<p>Melodex is a social media backend inspired by Instagram, providing features like posts, stories, interactions, and personalized content recommendations. Each user receives a unique feed based on their interests and activity patterns.</p>
<div>
  <h2><img width="30" height="30" alt="image" src="https://github.com/user-attachments/assets/03e14750-4e6f-4353-a1f7-dfb44e915b23"/> Features</h2>
  <ul>
    <li>Personalized feed: users see posts based on their interests</li>
    <li>Create posts image + caption + description</li>
    <li>Create stories</li>
    <li>Like & unlike posts</li>
    <li>Save & unsave posts</li>
    <li>Commenting system</li>
    <li>User authentication (sign-up & login)</li>
    <li>more...</li>
</ul>

  <div>
    <h3>1: clone repo</h3>
    
```     
 git clone https://github.com/AminAhmadi070807/melodex.git
```

  <h3>2: entering the project</h3>
    
```
cd ./melodex
```

  <h3>3: installation package project</h3>
    
```
npm i
```

  <h3>4: start the server</h3>
    
```
npm run dev
```

  <h3>5: entering this address to broswer</h3>
    
```
http://localhost:3000/api/docs
```
